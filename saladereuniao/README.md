# Gerenciador de Salas de Reuniões - backend

## Build

```bash
mvn install
```

## Build clean

```bash
mvn clean package
```
   
## Run

```bash
mvn spring-boot:run
```

## Tecnologias

* Java com Spring boot
* Dependências 
  *  Spring Data JPA
  *  H2 Database
  *  Lombok
  *  Spring web
*  Dependência de desenvolvimento
  *   Spring boot DevTools
